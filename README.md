# customer-churn-predictor
An end-to-end Customer Churn Prediction system built using TensorFlow, Keras, and Streamlit. The project includes data preprocessing, feature engineering, label encoding, one-hot encoding, feature scaling, ANN model training, and an interactive web application for real-time churn prediction.

Tech Stack:
• Python
• Pandas
• NumPy
• Scikit-learn
• TensorFlow / Keras
• Streamlit

Features:
• Data Cleaning & Preprocessing
• Label Encoding & One-Hot Encoding
• Feature Scaling
• ANN-based Binary Classification
• Customer Churn Probability Prediction
• Interactive Streamlit Dashboard
